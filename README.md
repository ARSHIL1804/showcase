- Install Docker Desktop
- Open this repository in vscode
- Install Dev container and Docker Extention in Vscode
- Press Ctrl+Shift+P -> Search for "Rebuild and Reopen in Container" and     press Enter
- Wait for all processes to complete



- Backend 
  - go to showcase-backend
  - run './mvnw spring-boot:run'

- Frontend
  - go to showcase-frontend
  - run 'npm run start'
